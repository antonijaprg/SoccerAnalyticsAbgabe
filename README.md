# SoccerAnalyticsAbgabe

## Setup

### Windows

```pwsh
python -m venv .venv
python -m pip install -U pip
. .\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

### macOS & Linux

```bash
python -m venv .venv
python -m pip install -U pip
. .venv/bin/activate
pip install -r requirements.txt
```
